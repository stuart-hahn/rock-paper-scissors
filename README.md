# Rock Paper Scissors Game

This is a simple Rock Paper Scissors game implemented with HTML, CSS, and JavaScript. The game is played against the computer, and the first to reach 5 points wins.

## Features

- Responsive and accessible UI for selecting Rock, Paper, or Scissors.
- Real-time game results and score updates displayed on the screen.
- Preserves the history of all played rounds.
- Clean and modular code structure following best practices.

## How to Play

1. Select Rock, Paper, or Scissors by clicking one of the buttons.
2. The game displays the results of each round and updates the scores.
3. The first to 5 points wins the game.

## Installation

1. Clone this repository:
    ```
    git clone https://github.com/yourusername/rock-paper-scissors.git
    ```
2. Open `index.html` in your browser.
